Дат је низ позитивних целих бројева. Написати функцију која одређује дужину најкраћег подниза узастопних елемената таквих да им је збир већи или једнак од задатог броја K.  
(https://leetcode.com/problems/minimum-size-subarray-sum/)

Пример:
```
Улаз                Излаз
K = 7
[2, 3, 1, 2, 4, 3]  2
```
