Дато је **N** књига и број страница сваке од њих.
Имамо **M** студената које треба распоредити да читају књиге при чему сваки од њих мора да чита бар једну књигу.
Уколико неки студент чита више књига мора да чита узастопне књиге тј. не може да чита нпр. књиге 1 и 3 без да чита и књигу 2.
Написати функцију која враћа минималан могући број страница које ће студент који чита највише страница морати да чита или −1 ако решење не постоји.  
(https://www.interviewbit.com/problems/allocate-books/)

Пример:
```
Улаз                Излаз
M = 2
[12, 34, 67, 90]    113
```
